# 📊 Customer Churn Analysis

## 📌 Project Overview
Customer churn is a major challenge for subscription-based businesses.  
This project analyzes customer demographics, service usage, contract types, and payment methods to uncover insights that help reduce churn and improve retention.

---
## ❓ Problem Statement
Telecom companies face high churn rates, especially among short-tenure customers and those on flexible contracts.  
The goal of this project is to **identify the key factors contributing to churn** and provide **data-driven recommendations** for customer retention.

## 📂 Dataset Information
- **Dataset:** Telco Customer Churn dataset (Kaggle)  
- **Rows:** ~7,000 customer records  
- **Columns:** Customer demographics, services subscribed, contract type, payment method, monthly charges, tenure, and churn status  
- **Target Variable:** `Churn` (Yes/No)

📊 Results
Overall churn rate: ~26.5%

Month-to-month contracts churn ~43% vs. 3% for two-year contracts

Customers with tenure <12 months account for ~50% of churn cases

Electronic check users churn at ~45% vs. 15–20% for credit card/bank transfer users

Customers paying >$70/month churn at ~40% vs. 15% for <$40

📈 Visualizations include bar charts, heatmaps, and distribution plots that highlight churn patterns.
